# Mock Payment Service

## Available commands:

Install dependencies:

```bash
make install
```

Run application:

```bash
make run
```

Run tests:

```bash
make test
```

Run lint:

```bash
make lint
```

Run migrations:

```bash
make migrate
```

### Docker

Up services:

```bash
make up
```

Down services:

```bash
make down
```

Logs:

```bash
make logs
```
